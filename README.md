# Akan Name Generator

## Description

A web application that calculates a user's Akan name based on their birthdate and gender.

Akan names are traditional Ghanaian names assigned based on the day of the week a person is born.

This app calculates the day of the week using the birthdate, then assigns the corresponding Akan name.

### Example

- Birthdate: 2026-02-03, Gender: Female → Akan Name: Akosua
- Birthdate: 1990-06-15, Gender: Male → Akan Name: Kofi

## Features

- Input form with date and gender
- Validation for day, month, and gender
- Calculation of day of the week
- Display Akan name
